2.2 Namespace
    • You can add a Frame to a Namespace if it’s not anonymous.
    • You can ask a Namespace for a Frame, if you know its name
    • Have a way to create an example Namespace outside of any test classes. This namespace should be interestingly populated. Add to it example frames as you proceed through the exercises
    • A Namespace can be asked if it’s empty

2.5 Have at least these objects in your example Namespace:
    • Party('The Bank')
    • Party('Government')
    • Party('Tech firm')
    • Book('European Book')
    • Book('Asian Book')
    • Book('American Book')
Arrange so that Party(‘The Bank’) owns all these Books.
